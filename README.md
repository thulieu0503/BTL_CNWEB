# Web quản lí cửa hàng bán đồ ăn vặt

## Mô tả
Đây là hệ thống website hỗ trợ quản lí cửa hàng bán đồ ăn vặt, bao gồm quản lí sản phẩm, khách hàng, đơn hàng, kho hàng, thanh toán và thống kê doanh thu.

## Chức năng chính
- Quản lí tài khoản và phân quyền
- Quản lí danh mục sản phẩm
- Quản lí sản phẩm
- Quản lí khách hàng
- Quản lí giỏ hàng và đặt hàng
- Quản lí đơn hàng
- Quản lí kho hàng
- Quản lí nhà cung cấp
- Quản lí khuyến mãi
- Quản lí thanh toán
- Thống kê và báo cáo

## Đối tượng sử dụng
- Quản trị viên
- Nhân viên bán hàng
- Nhân viên kho
- Khách hàng

## Quy trình nghiệp vụ
1. Khách hàng chọn sản phẩm và đặt hàng
2. Hệ thống tạo đơn hàng
3. Nhân viên xác nhận đơn
4. Kho chuẩn bị hàng và cập nhật tồn kho
5. Giao hàng và thanh toán
6. Hệ thống lưu lịch sử đơn hàng và thống kê doanh thu

## Database
Các bảng chính:
- roles
- users
- categories
- products
- suppliers
- customers
- orders
- order_details
- payments
- promotions
- inventory_logs

## Mục tiêu
Hệ thống giúp quản lí cửa hàng chính xác, nhanh chóng, tiết kiệm thời gian và nâng cao hiệu quả kinh doanh.
